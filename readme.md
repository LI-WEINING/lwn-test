# readme
+ 项目文档

## git init
1. 把当前目录初始化为版本库
2. 会在目录下生成一个隐藏文件.git

## git add 文件名
1. 把当前下文件目录下文件提交到暂存区域
2. git add readme.md 把这个文件提交到暂存区
3. git add . 当前目录所有变动提交到展存区域

## git status
1. 查看当前文件目录状态（新增、删除、修改）

## git commit -m '提交注释文本'
1. 把暂存区的内容提交到本地仓库

## 本地涉及到的三个部分
1. 工作区域（实际文件）
2. 展存区域
3. 本地仓库

## git log
1. 查看操作日志 

## git reflog
1. 查看操作日志（简单版）

## git diff 文件变更信息
1. 查看文件变更信息
